# Assembly

A word-guessing game where you try to save Assembly language from disappearing — one letter at a time. Guess wrong too many times and Assembly is lost forever!

## How It Works
- A random word is selected at the start
- Guess letters by clicking or typing
- Each wrong guess brings Assembly closer to extinction
- Win by guessing the full word before running out of attempts

## Features
- Interactive letter selection
- Visual feedback for correct/incorrect guesses
- Win/lose states with confetti celebration on victory
- Clean, responsive UI

## Tech Stack
- React
- JavaScript
- CSS
- react-confetti
- clsx
